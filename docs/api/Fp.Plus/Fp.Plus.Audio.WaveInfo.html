Parameters

subChunk1Size int

16 for PCM. This is the size of the rest of the Subchunk which follows this number.

audioFormat short

PCM = 1 (i.e. Linear quantization) Values other than 1 indicate some form of compression.

numChannels short

Mono = 1, Stereo = 2, etc.

sampleRate int

8000, 44100, etc.

byteRate int

== SampleRate * NumChannels * BitsPerSample/8.

blockAlign short

== NumChannels * BitsPerSample/8 The number of bytes for one sample including all channels.

bitsPerSample short

8 bits = 8, 16 bits = 16, etc.

extraParamSize short

if PCM, then doesn't exist.

extraParams ReadOnlyMemory<byte>?

space for extra parameters.

subChunk2Size int

== NumSamples * NumChannels * BitsPerSample/8 This is the number of bytes in the data. You can also think of this as the size of the read of the subchunk following this number.

fact ReadOnlyMemory<byte>?

"fact" chunk body.
